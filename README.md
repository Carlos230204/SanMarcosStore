# SanMarcos Store

Aplicación Android desarrollada utilizando Jetpack Compose y Material 3.

## Descripción

SanMarcos Store es una aplicación de ejemplo que implementa una arquitectura moderna basada en Jetpack Compose, navegación mediante Navigation Compose y un diseño siguiendo Material Design 3.
